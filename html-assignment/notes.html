Html Tags

The Anchor Tages
We use the opening and closing of(a)for link to pages, files, etc.
The Body Tags
We make use of the html part(body) because it gives out the meaningful piece of information within the body, such as headings, tables,etc.
The Heading Tags
We have header indicators(h) opening and closing tags. lt helps to organise our website.
The image Tags
We use the(img) to show or embed images in the documents page.
The ordered list Tags
We make use of (ol) for an ordered list when listing items.

Tag Explanations

When creating all these tags in your text, always remember it has opening and closing Tags; for example {<} opening tags and {</>} closing tags.
